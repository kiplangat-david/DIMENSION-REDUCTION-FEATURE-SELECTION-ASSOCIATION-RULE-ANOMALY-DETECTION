##4. ANOMALY DETECTION##

The R ‘anomalize’ package enables a workflow for detecting anomalies in data. The main functions are: - time_decompose()- for time series decomposition - anomalize()- for Anomaly detection of remainder - time_recompose()- for Anomaly lower and upper bound transformation

##Decomposing##

We will use time_decompose() function in anomalize package. We will use stl method which extracts seasonality

df %>%
time_decompose(Sales, method = 'stl', frequency = 'auto', trend = 'auto') %>%
anomalize(remainder, method = 'gesd', alpha = 0.1, max_anoms = 0.5) %>%
plot_anomaly_decomposition(ncol = 3, alpha_dots = 0.7)

##Recomposing## -The time_recompose() function is used to generate bands around the “normal” levels of observed values

df %>%
